# Discrete Mathematics

## Motivation & Introduction to Propositional Logic

---

## 1. What is Logic?

Logic is the **science of reasoning**. It provides a structured way to think clearly and make valid conclusions from given information.

In mathematics, logic helps us:

* Analyze statements
* Determine whether they are true or false
* Build valid arguments (proofs)

### Why this matters

Without logic, mathematics would be based on guesswork. Logic ensures that every conclusion follows strict, well-defined rules.

### Example

Consider the statement:

> "For every positive integer n, the sum of integers from 1 to n is
> n(n + 1) / 2"

Logic helps us:

* Understand the statement
* Verify if it's correct
* Prove it rigorously

---

## 2. Why is Logic Important in Mathematics?

Logic is the **foundation of all mathematics**.

### Key roles of logic:

* Helps in constructing **proofs**
* Distinguishes between **valid and invalid arguments**
* Ensures mathematical results are **universally true**

### Important concept:

* If a statement is proven true → it becomes a **Theorem**

### Real-world reasoning example

**Knights and Knaves Puzzle:**

* Knights → always tell the truth
* Knaves → always lie

Given:

* Alice says: *"Bob is a Knight"*
* Bob says: *"We are of opposite types"*

Using logical reasoning, we can determine who is lying and who is telling the truth.

This type of problem is solved using **propositional logic**.

---

## 3. What is Propositional Logic?

The term *Propositional Logic* consists of two parts:

* **Proposition**
* **Logic**

### Definition:

Propositional logic is a branch of logic that deals with **propositions and their truth values**.

It focuses on:

* Whether a statement is **TRUE or FALSE**
* Combining statements using logical operations

---

## 4. What is a Proposition?

A **proposition** is a:

> Declarative sentence that is either TRUE or FALSE, but not both.

### Key properties:

* Must **state a fact**
* Must have a **definite truth value**
* Cannot be ambiguous

### Examples:

| Statement                          | Truth Value                       |
| ---------------------------------- | --------------------------------- |
| Dhaka is the capital of Bangladesh | TRUE                              |
| Water froze this morning           | Depends (but still a proposition) |
| 1 + 1 = 2                          | TRUE                              |

---

## 5. Sentences Which Are NOT Propositions

A sentence is **not a proposition** if:

* It is a question
* It is a command
* It depends on unknown variables
* It expresses emotion or request

### Examples:

| Sentence                  | Reason       |
| ------------------------- | ------------ |
| What time is it?          | Question     |
| x + 1 = 2                 | Depends on x |
| Send your resume          | Command      |
| Please allow me a day off | Request      |
| Fetch my umbrella!        | Command      |

## Final Summary

* **Logic** → helps us reason correctly
* **Proposition** → a statement with TRUE or FALSE value
* **Propositional Logic** → studies these statements and their relationships
* Not all sentences are propositions
* Mathematics relies heavily on logic to build **proofs and theorems**

---

## Quick Mental Checklist

To check if something is a proposition:

Ask:

1. Is it a complete statement?
2. Does it declare something?
3. Can it be clearly TRUE or FALSE?

If all answers are **YES → it's a proposition**
